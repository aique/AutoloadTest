		static method parse (line 31)
	
 
	
	

Parsea el fichero de configuración y devuelve un array asociativo  con los valores encontrados en un formato interpretable por  la aplicación.


El array devuelto tendrá el siguiente formato:
  • ("entorno1" => ("atributo1" => "valor1", "atributo2" => "valor2"), "entorno2" => ("atributo3" => "valor3") ... )

	
    
				
  • return: Array asociativo con los datos de configuración establecidos.
