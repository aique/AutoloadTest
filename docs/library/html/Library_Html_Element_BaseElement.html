		getValue (line 129)
	
 
	
	

Devuelve el valor del atributo value.


Es comúnmente utilizado para conocer el valor que el usuario  ha introducido en un campo. Sin embargo, no todos los elementos  HTML funcionan de esta manera.
Por ejemplo, un checkbox tendrá valor 1 si está marcado  y 0 si no lo está. Cada elemento específico por tanto deberá  sobreescribir este método para devolver su valor de la manera  oportuna.

	
    
				
  • return: Cadena de texto con el valor del atributo value. Si no se          ha establecido todavía, devolverá una cadena vacía.
