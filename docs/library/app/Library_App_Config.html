Class Library_App_Config

Description
Description | Vars (details) | Methods (details)

Clase que gestiona el contenido del fichero de configuración, así como el entorno en el que se está ejecutando la aplicación.

Cuando se instancia un objeto de esta clase se realiza un parseo del fichero de configuración de la aplicación. A partir de ese momento, cada una de las variables establecidas en el mencionado fichero serán accesibles utilizando los métodos de esta clase.
