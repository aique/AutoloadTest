Class Library_App_Dispatcher

Description
Description | Methods (details)

Se encarga de delegar en el controlador adecuado una petición realizada sobre la aplicación.

Se instancia dentro del bootstrap como una de las primeras acciones que se realiza dentro del proceso de atención de una petición.
